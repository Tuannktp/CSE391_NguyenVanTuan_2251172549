Phần 1: Content-Box vs Border-Box

Mỗi hộp có: width: 300px, padding: 20px, border: 5px solid

Hộp 1: Content-Box

box-sizing: content-box (mặc định)

Chiều rộng = width ONLY (300px)

Nội dung bên trong

Hộp 2: Border-Box

box-sizing: border-box

Chiều rộng = width (300px) bao gồm padding + border

Nội dung bên trong

📋 Hướng dẫn đo đạc:

  1. Mở DevTools (F12)
  2. Chọn Element tab
  3. Click vào mỗi hộp
  4. Cuộn xuống tab Computed/Styles
  5. Tìm phần Box Model diagram
  6. Ghi lại chiều rộng thực tế của mỗi hộp
  7. Chụp screenshot Box Model diagram

Phần 2: Layout 3 Cột (Border-Box)

Container: 1000px (với border-box, layout đúng)

Cột Trái

Sidebar

250px + padding 15px

Cột Giữa

Main Content

500px + padding 20px

Đây là phần nội dung chính của trang web. Layout này sử dụng border-box nên tổng chiều rộng của 3 cột vừa khít 1000px.

Cột Phải

Ads/Sidebar

250px + padding 15px

Tính toán WITH border-box:
(250 + 15×2) + (500 + 20×2) + (250 + 15×2) = 280 + 540 + 280 = 1100px... ❌ Không vừa! Cần điều chỉnh hoặc dùng display: flex

Phần 2B: Layout 3 Cột (KHÔNG Border-Box - để so sánh)

Container: 1000px (KHÔNG border-box, layout sẽ vỡ)

Cột Trái

Sidebar

250px + padding 15px

Cột Giữa

Main Content

500px + padding 20px

Không dùng border-box nên padding sẽ THÊM VÀO chiều rộng. Tổng > 1000px → layout vỡ!

Cột Phải

Ads/Sidebar

250px + padding 15px

Tính toán WITHOUT border-box:
(250 + 15×2) + (500 + 20×2) + (250 + 15×2) = 280 + 540 + 280 = 1100px
❌ TOTAL = 1100px > 1000px → CỘT PHỦ BỊ ĐẨYDÂY XUỐNG!

Phần 3: Layout 3 Cột (Flexbox - Giải pháp tốt hơn)

Container: 1000px (dùng flexbox + border-box)

Cột Trái

Sidebar - 250px

Flexible layout tự động căn chỉnh

Cột Giữa

Main Content - flex: 1

Flexbox tự động chia khoảng trắng đều. Không cần tính toán phức tạp!

Cột Phải

Ads - 250px

Layout responsive và sạch

Flexbox: 250 + flex:1 + 250 = tự động căn chỉnh (bản hiện đại hơn)
